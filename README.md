Работа с Git

1. Клонирование репозитория
   Выполнить в любом удобном месте проводника Windows следующую команду в терминале:

git clone demo2026@192.168.11.86:/home/demo2026/demo26.git
Попросит ввести пароль - вводим пароль "demo26". После этого откройте данный склонированный репозиторий в PyCharm (как
обычную папку):

2. Создание рабочей ветки
   Далее необходимо открыть склонированный проект в PyCharm, и далее открыть в нём терминал (желательно Git Bash или
   PowerShell чтобы не было проблем)

Далее нужно создать ветку и переключиться на неё с названием своей фамилии (на английском):

git switch -c ВашаФамилия
Пример:

git switch -c Ivanov
После переключения терминал показывает:

~\Documents\demo26 git:[ВашаФамилия]

3. Проверка веток (на всякий случай)
   git branch
   Пример вывода (где звёздочка - там означает то, что вы находитесь в текущий момент на данной ветке):

* ВашаФамилия
  master

4. Добавление файлов проекта
   Необходимо добавить файлы проекта в папку demo26. Далее произвести команду ниже:
   git add .
   Возможны предупреждения:

5. Создание коммита
   git commit -m "project added"


6. Отправка на сервер
   Далее необходимо произвести команду по отправке данных (вашего проекта) на сервер:
   git push -u origin ВашаФамилия
   При выполнении запрашивается пароль: вводим пароль "demo26"
